## 概述

上海证券交易所交易网关（TDGW）STEP接口规格说明书（互联网交易平台）V1.14（2025年11月版）。

通过 TDGW 对接互联网交易平台系统，提供实时交易流接口。TDGW 通过交易业务单元（PBU）登录并接入交易系统。

## 交互机制

### 会话建立
- OMS 发起 TCP 连接 → 发送 Logon 消息
- TDGW 返回 Logon（成功）或 Logout（失败）
- 心跳间隔：[5, 60]秒，以 TDGW 返回为准
- 5个心跳间隔未收到消息则关闭连接

### 登录PBU与订阅PBU
- 业务交易单元需与登录交易单元同属一家机构
- 订阅PBU也需同属一家机构
- 执行报告分区信息 ExecRptInfo 提供 PBU 和分区列表

### 消息类型
| MsgType | 说明 |
|---------|------|
| D | 新订单申报（New Order Single） |
| F | 撤单申报（Order Cancel） |
| 8 | 执行报告/申报响应（Execution Report） |
| 9 | 撤单失败（Order Cancel Reject） |
| R | 询价请求（Quote Request） |
| S | 报价（Quote） |
| AJ | 报价回复（Quote Response） |
| U006 | 网络密码服务申报 |
| U008 | 密码服务申报响应 |
| U108 | 执行报告分区信息 |
| U109 | 平台状态 |

## 基金通询价交易流程

1. 询价方发起询价请求 → 系统转发给报价方
2. 报价方进行报价（可撤销后重报）
3. 询价方接受报价（报价回复）→ 系统撮合成交
4. 询价方可撤销询价，报价方可撤销报价
