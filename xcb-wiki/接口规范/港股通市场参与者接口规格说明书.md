## 概述

上海证券交易所港股通市场参与者接口规格说明书 v1.08（2024年8月版），通过 EzSTEP 协议接入上交所港股通系统。

## 业务类型

| reqid | 说明 |
|-------|------|
| HKN | 港股整手订单申报 |
| HKO | 港股零股订单申报（碎股订单） |

- 产品集编号（setid）：107
- 数据广播类型（bcasttype）：19（成交回报和自动撤单回报）

## 消息类型

| MsgType | 说明 |
|---------|------|
| D | 申报（New Order Single） |
| F | 申报撤单（Order Cancel） |
| 8 | 申报响应/执行报告（Execution Report） |
| 9 | 申报撤单失败（Order Cancel Reject） |

## 申报消息字段

| 字段 | 说明 |
|------|------|
| SecurityID | 港股代码，5位左补空格成6位 |
| Price | 价格，单位港元 N11(3) |
| OrderQty | 订单数量 N10 |
| Side | 1=买, 2=卖 |
| OrdType | 2=限价（零股无意义） |
| MaxPriceLevels | 1=竞价限价盘, 0=增强限价盘（默认） |
| TimeInForce | 0=当日有效(GFD), 暂不支持FOK |

## 港股通盘后文件接口

### 过户数据 hghXXXXX.dbf
文件名中 XXXXX 表示业务交易单元号，闭市后发送。压缩为 hghXXXXX.zip。

| 字段 | 说明 |
|------|------|
| cjbz | HKN=整手, HKO=零股 |
| bs | B=买, S=卖 |

### 成交单据文件 tjss04XXXXX.txt
由中国结算上海分公司生成，上交所转发。压缩为 tjs04XXXXX.zip。

包含：会员名称、证券代码、买卖方向、成交数量/价格/金额、印花税、成交编号、交收日期等。

## 行情文件接口

### 实时行情 mktdt04.txt
通过交易所行情信道发送，文件结构：文件头 + 文件体 + 文件尾。

| MDStreamID | 数据类型 | 说明 |
|-----------|---------|------|
| MD401 | 港股通证券实时行情 | 五档行情、成交信息 |
| MD404 | VCM数据 | 市调机制（冷静期）信息 |
| MD405 | CAS数据 | 收市竞价交易时段信息 |
| MD406 | POS数据 | 开市竞价交易时段信息 |

### 交易盘实时状态 trdses04.txt
上交所对港股通订单的额外控制（不同于港交所行情状态）。

| MDStreamID | 数据类型 | 说明 |
|-----------|---------|------|
| MD402 | 实时额度数据 | 每日初始额度、日中剩余额度、额度状态 |
| MD403 | 产品状态数据 | 整手/零股买入卖出限制 |

### 港股基础信息 reff04MMDD.txt
港交所生成，上交所转发，交易日开市前发送。

### 参考汇率 exra04MMDD.txt
中国结算上海分公司生成，交易日开市前发送。

### 最小价差文件 zxjcMMDD.txt
港交所生成，记录各价位表的价格区间和价差组别。
